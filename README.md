# mathplay

Play a simple math game. Requires `python` >= 3.9. On windows use Windows Terminal to get best user experience.

- User language is german
- Only uses small numbers (<10)
- Only support for following math operations:
  - add

Start the game with:

```python
python mathplay.py
```
